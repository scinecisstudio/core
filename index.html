FEATURES

Everything the app gives you, clearly organized.

Fast App Drawer

Launch apps instantly with typo-tolerant fuzzy search, optional alphabet scrollbar, and support for pinning and reordering your most used apps.

Quick Tiles

Add favorite contacts, one-tap system setting shortcuts, weather data, and calendar events with countdown info for what is next.

Widget Board

Add Android widgets, move them on a grid, resize from corners, and remove widgets in edit mode with visual handles.

App Privacy Controls

Hide apps from normal view and lock selected apps with passcode authentication, plus biometric unlock support when available.

Launcher Gestures

Enable swipe down for notifications and double tap to sleep (with accessibility service) for faster one-hand interaction.

Backup & Restore

Export settings to JSON and restore later. Core detects required permissions before importing and guides users through missing access.

PERMISSIONS

Transparent permission usage.

Package Visibility

Used by the launcher to discover installed apps, search them, and support launcher management flows.

Contacts

Used for contact photos, starred contacts, favorite contacts in Quick Tiles, and contact search actions.

Calendar

Used to display today's events and event countdown in Quick Tiles.

Biometric

Used for optional biometric confirmation in app lock flows.

Accessibility

Used for gestures (swipe notifications, double tap sleep) and stronger app lock behavior.

Notification Access

Used to show notification badges on app icons.

Overlay Permission

Used for optional internet speed indicator in the status bar area.

Photos and Media

Used locally for wallpaper-based appearance tuning, such as automatic status bar and drawer text color, on supported Android versions.

Internet

Used for weather data integration through OpenWeather.

FAQ

Common questions

Does Core support widgets?

Yes. You can add, move, resize, and remove widgets on a dedicated widget page.

Can I hide and lock apps?

Yes. Core supports hidden apps and app lock with passcode and biometric options.

Can I back up my settings?

Yes. Core exports launcher settings to JSON and supports restoring from backup.

Is it customizable?

Core includes theme control, grid size, icon size, app categories, gestures, and more.
